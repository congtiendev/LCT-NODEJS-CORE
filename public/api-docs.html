Authentication

POST /auth/register
Register a new user account

Request Body:

{ "name": "John Doe", "email": "john@example.com", "password": "password123" }

Validation Rules:

  • name: 2-100 characters (required)
  • email: valid email format (required)
  • password: minimum 8 characters (required)
